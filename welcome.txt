Добро пожаловать на неофициальный сайт ДНЗ 144 города Киев (Ромашка).
Этот сайт создан как некоммерческий проект целью которого является:
- Сделать бухгалтерию группы максимально прозрачной и легко доступной каждому родителю.
- Облегчить коммуникацию между членами группы и сохранять результаты решений к которым пришла группа в следствии дискуссий, что бы не поднимать темы дважды.
- Сделать систему голосования надежней. Принимается только один голос от одного ребенка из группы, результаты сохраняются в одном месте, голосование является закрытым.
- Собрать самую важную и часто востребованную информацию в одном месте.

Конфиденциальность:
1) Незарегистрированный пользователь имеет доступ только к самой общей информации (Место расположения ДНЗ, контакты и т.д);
2) Регистрация без привязки к ребенку дает возможность просматривать результаты голосований;
3) Привязка пользователя к ребенку дает все преимущества сервиса.
4) Привязка к группе осуществляется через ребенка. При этом данные ребенка (имя и фамилия) отображаються только пользователям которые привязаны к ребенку.

Сайт содержит следующие особенности:
Сервисы:
1) Открытые вопросы. Аля блог в котором можно вести дискуссии касательно определенной темы. Любой пользователь имеющий привязку к группе может открыть свой вопрос группе.
2) Закрытые вопросы. Содержат решения к которым пришли участники группы. Право закрывать вопрос имеет только персонал сайта.
3) Голосования. Вопросы отдельно — голосования отдельно.
Бухгалтерия:
1) Активные фонды. Содержит список всех активных сборов с детальной информацией.
2) Закрытые фонды. Фонды, которые были закрыты по тем или иным причинам.
3) Благотворительные фонды (в разработке). Будут открываться фонды целью которых будет собрать определенную сумму средств для приобретения заранее обговоренного товара. Не будет являться обязательными для всех участников группы.
4) Отчеты (в разработке). Будет разработана система с помощью которой можно будет получить информацию в таблице по своим балансам, движениям средств и т.д.

Приятного пользования! =)
